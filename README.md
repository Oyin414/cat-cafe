# About CAT CAFE
This is a cat cafe website!





**Link to project:** https://oyin414.github.io/cat-cafe/


## How It's Made:

**Tech used:** HTML, CSS, JavaScript, webpack

This cat cafe website uses ES6 modules and npm to bundle my JavaScript Code.This website was made so that I could practice using the
ES6 Modules and so I could use webpack to bundle my files.The use of ES6 Modules means all the code written in the individual modules are
locally scoped which allow my variables to be private , without ES6 Modules all my code would share the same  global scope which would lead 
to countless errors.



## Lessons Learned:
 
During this project I learnt to use import and export different modules so that different parts of my work had their own modules. I 
learnt to use webpack to bundle my JavaScript and configure webpack to handle my CSS and HTML files so that I could generate the entire 
contents of a webpage . While using weback I struggled initially with usingg DOM manipulation and exporting and importing the right files
but after practicing with it for a few hours I managed to figure out how to get the changes i made to the DOM to show up on the webpage.
